■動作内容
　24時間以内に更新されたはてなブログの記事に星を追加します。
　更新日時の新しい順で処理します。


■セットアップ方法

1.Install.batを実行してください。

2.Hatena.addStar.batを編集してください。

　下記の項目に値を設定してください。
　
  □必須
  ・はてなユーザ名：USER
  ・はてなパスワード：PASS
  の"="の右側に
  あなたのユーザ名（ID)とパスワードを設定してください。
  
  □オプション
  ・星をつけるサイト数：NUM⇒10～100の範囲で件数を設定してください。
  ・Google検索キーワード：KEYWORD⇒特定のキーワードにHITするサイトのみ
  　　　　　　　　　　　　　　　　対象にしたい場合、設定してください。


■起動方法
Hatena.addStar.batを実行してください。


実行結果は[日付.log]ファイルに出力されます。
ログで行末にres:{○○,name="ユーザー名",○○} とあるものは星追加成功になります。


■動作要件
node.js ver8以上

Node.js Windowsへの導入方法（Masayuki-M様）
https://qiita.com/Masayuki-M/items/840a997a824e18f576d8

起動に失敗する場合node.jsのパスが通っているか確認してください。
